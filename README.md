# PlacePicker Web App

## Stack

- Backend --> Nodejs
- Frontend --> Angular
- Data --> dummy data in backend
- Webapp operations --> fetching data from bakcend API (through HTTP requests)

## Angular Concepts

- Sending/receiving http requests GET, PUT, DELETE
- Handling errors globally
- Loading state
- tap(), pipe(), map()
- Interceptors
- Data handling
